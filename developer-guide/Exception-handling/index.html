异常处理

多用try/expect，既可以增加模块的稳定性也可以减少用户的判断。 需要注意的是对于用户输入ctrl+c这样的键盘中断信号，不需要模块自己捕获，相反还需要弹出交给MultiProxies来处理。比如:

try:
    Do somethings
except KeyboardInterrupt:
    raise
except Exception,e:
    color.echo(e, RED)
